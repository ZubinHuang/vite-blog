# 响应式原理总结

* 利用 Proxy 实现了对数据访问和修改的劫持 弥补了 Object.defineProperty 的不足
* 响应式的核心实现就是通过数据劫持
  * 在访问数据的时候执行依赖收集
  * 在修改数据的时候派发通知
  * 收集的依赖是副作用函数 数据改变后就会触发副作用函数的自动执行
* 把数据变成响应式 是为了数据在变化的时候自动执行一些逻辑
* 组件的渲染中就是让组件访问的数据一旦被修改 就会自动触发组件的重新渲染 实现数据驱动
